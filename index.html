Bowler.js

Bowler.js is a Model / View framework for the gentleman in you.

Use it

Bowler is super simple to use. Just include the file then...

var app = Bowler();
app.extend({title : "hello"});
app.render('<p>{{title}}</p>');
app.bind('body');

Child Models

Whats the use of only having one model. Bowler supports child models.

var page = app.Model('page', { world: "universe" });
var pageView = app.View('page', "<b>{{universe}}</b>");
//changing the main models template
app.render("<h1>{{title}}{{{child "world"}}}</h1>");
